# To-Do App

A minimal and fast to-do application to manage daily tasks with focus and simplicity.

## Features
- Add, edit, and delete tasks  
- Mark tasks as completed  
- Clean and distraction-free UI  
- Responsive design  

## Tech Stack
- Frontend: React  
- Styling: Tailwind CSS  
- State Management: Local State / Context API  

## Getting Started
```bash
git clone <repo-url>
cd todo-app
npm install
npm run dev
```
